# Installation

`npm install`

# Start the presentation

`npm start`

# Keyboard shortcuts

- `s`: speaker notes

Navigation:

- `ESC`: toggle overview
- `SPACE`: Forward
- `Arrow Keys`: navigate in direction of arrow
